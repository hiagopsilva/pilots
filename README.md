## Como Começar

Siga estas instruções para configurar e rodar o projeto em seu ambiente local.

### Passo 1: Instalação

Clone o repositório e instale as dependências.

```bash
# utilizando npm
npm install

# ou utilizando Yarn
yarn install
```

### Passo 2: Execução 

```bash
# utilizando npm
npm start

# ou utilizando Yarn
yarn start
```

### Passo 2.1: Execução - Android

```bash
# utilizando npm
npm run android

# ou utilizando Yarn
yarn android
```

### Passo 2.2: Execução - iOS

```bash
# utilizando npm
npm run ios

# ou utilizando Yarn
yarn ios
```
